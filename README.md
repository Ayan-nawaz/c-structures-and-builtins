# c-structures-and-builtins

A collection of beginner-friendly C programs demonstrating structures, typedef, built-in functions, and user-defined data handling. This repository helps learners understand how structures organize data efficiently in C.

## Programs Included

- `arr-of-structures.c` – Demonstrates arrays of structures.
- `books-structure.c` – Manages book information using structures.
- `built-in-fun.c` – Examples of commonly used built-in C library functions.
- `structures-input.c` – Takes user input using structures.
- `type-deff.c` – Demonstrates the use of `typedef` with structures.

## Topics Covered

- Structures
- Arrays of Structures
- User Input with Structures
- `typedef`
- Built-in Library Functions
- User-Defined Data Types

## Language

- C Programming

## Compiler

Compatible with any standard C compiler:

- GCC
- Code::Blocks
- Turbo C
- VS Code (GCC)

## How to Run

1. Clone this repository.
2. Open any `.c` file.
3. Compile the program:

   ```bash
   gcc filename.c -o output
   ```

4. Run the executable:

   ```bash
   ./output
   ```

## Purpose

This repository is created for beginners who want to practice structures, typedef, and built-in functions while strengthening their C programming fundamentals.
